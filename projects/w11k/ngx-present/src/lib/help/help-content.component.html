Navigation

menu Click on the menu icon in the upper left corner to open the menu.
Via the menu you can navigate to the different modes like slide-by-slide, overview and presenter. You can also use the table of content to jump to specific sections, if the author of the presentation uses this feature.

Navigation in Slide Mode

Key Action
arrow_downward Arrow Down or Page Down Go to next slide
arrow_upward Arrow Up or Page Up Go to previous slide
arrow_forward Arrow Right Go to first slide of the next section
arrow_back Arrow Up Go to first slide of the current section
skip_previous Pos 1 Go to first slide
m Open menu
